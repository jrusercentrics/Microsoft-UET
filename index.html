Test – GTM Old Documentation Setup

In GTM, add two tags:

  1. Cookiebot CMP (Custom HTML tag with your Cookiebot script, blocking mode = auto).
  2. Microsoft UET (using GTM’s built-in UET template, with your tag ID).

Then open the browser console (F12) and run: 

window.uetq?.uetConfig?.enableAdStorage
  • Before consent: you may see true (granted by default) or nothing if Cookiebot blocks UET.
  • After accepting/rejecting marketing: Cookiebot should push an update.
